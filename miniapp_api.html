trueCORD Mini Apps API

Документация для мини-приложений trueCORD. Mini Apps открываются внутри канала и могут работать с сообщениями, пользователями, уведомлениями и локальным хранилищем через встроенный JavaScript API.

Версия 1.0.0

MiniApp API предоставляет мини-приложениям доступ к функциям платформы: отправка сообщений, уведомления через Dynamic Island, управление данными, приглашения в игры и реалтайм-подписки. Для безопасности API работает только внутри iframe, загруженного платформой.

Подключение API

Мини-приложение работает внутри <iframe> с атрибутом sandbox="allow-scripts allow-forms allow-popups allow-same-origin". Есть два способа обращаться к API:

Способ 1 — прямой вызов (если same-origin)

// Прямой доступ через window.parent (работает только если
// мини-апп хранится на том же домене что и trueCORD)
const API = window.parent.TES3API;

// Получить текущего пользователя
const me = API.getMe();
console.log(me.user.name); // 'Admin'

// Показать уведомление в Dynamic Island
API.notify('Ход сделан!', 'Ваш противник походил в шашках');

Способ 2 — postMessage (универсальный)

Используйте postMessage если приложение хранится на другом домене или вы хотите async/await интерфейс:

// Вспомогательный класс для работы с TES3API через postMessage
class TrueCordBridge {
  constructor() {
    this._pending = {};
    this._id = 0;
    window.addEventListener('message', (e) => {
      if (!e.data?.tes3 || !e.data.callId) return;
      const { callId, result } = e.data;
      if (this._pending[callId]) {
        this._pending[callId](result);
        delete this._pending[callId];
      }
    });
  }

  call(action, params = {}) {
    return new Promise((resolve) => {
      const callId = ++this._id;
      this._pending[callId] = resolve;
      window.parent.postMessage({ tes3: true, callId, action, ...params }, '*');
    });
  }
}

// Использование
const bridge = new TrueCordBridge();

const me = await bridge.call('getMe');
console.log(me.user.name);

await bridge.call('notify', {
  title: 'Игра найдена!',
  text: 'Соперник принял вызов',
  opts: { icon: '🎮' }
});
Совет: Сохраните код класса TrueCordBridge в начале своего мини-приложения. Это обёртка с нулевыми зависимостями, которая работает в любом браузере.

function TES3API.notify(title, text, opts?) Dynamic Island уведомление

Показывает уведомление через Dynamic Island. Если пользователь кликнет — срабатывает opts.onClick.

ПараметрТипОписание
titlestringЗаголовок уведомления
textstringТекст уведомления

API.notify('🎮 Ход противника', 'Петров сделал ход в шашках', {
  icon: '♟',
  dotColor: 'red',
});

function TES3API.toast(message, type?, duration?) Системное сообщение через DI

Краткое системное сообщение в Dynamic Island (не остаётся в истории уведомлений).

ПараметрТипОписание

getMe()

function TES3API.getMe() Текущий пользователь

Возвращает профиль авторизованного пользователя.

Возвращает: {ok, user: {id, name, avatar, validated, globalRole}}
const { user } = API.getMe();
// user = { id: 1, name: 'Admin', avatar: '', validated: true, globalRole: 'super_admin' }

getMembers()

function TES3API.getMembers() Участники сервера

Список всех участников текущего сервера с их статусами онлайн.

Возвращает: {ok, members: Member[]}
const { members } = API.getMembers();
const online = members.filter(m => m.online);
console.log(`Онлайн: ${online.length}`);

sendMessage(channelId, text)

async TES3API.sendMessage(channelId, text) Отправить сообщение

Отправляет сообщение в указанный канал от имени текущего пользователя. Требует верификации.

ПараметрТипОписание
channelIdnumberID канала
textstringТекст сообщения (макс. 4000 символов)
const r = await API.sendMessage(42, 'Я выиграл в шашках! 🏆');
if (r.ok) API.toast('Сообщение отправлено', 'ok');

inviteToGame(gameName, payload?)

async TES3API.inviteToGame(gameName, payload?) Пригласить в игру

Отправляет структурированное приглашение в игру через системное сообщение в текущем канале. Другие пользователи могут принять приглашение через acceptInvite().

ПараметрТипОписание
gameNamestringИдентификатор игры (напр. 'checkers')
payloadobject?Произвольные данные (параметры партии и т.д.)
Формат сообщения: [INVITE:gameName] {"key":"value"}
// Игрок 1: отправляет приглашение
await API.inviteToGame('checkers', {
  color: 'white',
  timeLimit: 600,
});

// Игрок 2: слушает сообщения, находит INVITE, принимает
const msgs = await API.getMessages(channelId);
const invite = msgs.messages.find(m =>
  m.text.startsWith('[INVITE:checkers]')
);
if (invite) await API.acceptInvite(channelId, invite.id);

storeGet / storeSet / storeDelete

async storeGet(key) / storeSet(key, value) / storeDelete(key) Хранилище данных

Персистентное key-value хранилище, привязанное к каналу мини-приложения. Данные сохраняются между сессиями. Значения автоматически сериализуются в JSON.

// Сохранить состояние игры
await API.storeSet('gameState', {
  board: boardArray,
  turn: 'black',
  moves: 14,
});

// Восстановить при загрузке
const saved = await API.storeGet('gameState');
if (saved.ok && saved.value) {
  const state = JSON.parse(saved.value);
  restoreGame(state);
}

subscribe(channelId, callback)

event TES3API.subscribe(channelId, callback) Подписка на сообщения

Подписывается на новые сообщения в канале. Callback вызывается при каждом новом сообщении через polling платформы. Работает только при прямом вызове через window.parent.TES3API.

Возвращает: функцию отписки () => void
const API = window.parent.TES3API;

// Подписка
const unsubscribe = API.subscribe(42, (msg) => {
  if (msg.text.startsWith('[ACCEPT:]')) {
    startGame(msg.userId);
  }
});

// Отписка при уничтожении компонента
window.addEventListener('beforeunload', unsubscribe);

Пример: Шашки (PvP)

Полный паттерн для реализации PvP-игры между двумя пользователями на платформе:

/**
 * ШАШКИ — trueCORD MiniApp
 * Полный пример PvP игры с invite/accept паттерном
 */

// 1. Инициализация
const bridge = new TrueCordBridge();
let gameState = { active: false, opponentId: null };
const me = await bridge.call('getMe');
const channelId = 42; // ID канала мини-апп

// 2. Отправить приглашение конкретному игроку
async function sendInvite(opponentName) {
  await bridge.call('notify', {
    title: '🎮 Приглашение',
    text: `@${opponentName} вы приглашены в шашки!`,
    opts: { icon: '♟' }
  });
  await bridge.call('inviteToGame', {
    gameName: 'checkers',
    payload: {
      inviterId: me.user.id,
      inviterName: me.user.name,
      target: opponentName,
    }
  });
}

// 3. Polling — ищем ACCEPT ответ
let lastMsgId = 0;
async function pollMessages() {
  const r = await bridge.call('getMessages', { channelId, limit: 10 });
  if (!r.ok) return;

  r.messages.forEach(msg => {
    if (msg.id <= lastMsgId) return;
    lastMsgId = Math.max(lastMsgId, msg.id);

    // Кто-то принял наше приглашение
    if (msg.text.startsWith('[ACCEPT:')) {
      gameState = { active: true, opponentId: msg.userId };
      startGame(msg.userId);
    }

    // Ход противника
    if (gameState.active && msg.userId === gameState.opponentId
        && msg.text.startsWith('[MOVE:')) {
      const move = JSON.parse(msg.text.slice(7, -1));
      applyMove(move);
      bridge.call('notify', {
        title: '♟ Противник ходит',
        text: `${msg.name} сделал ход`
      });
    }
  });
}
setInterval(pollMessages, 2000);

// 4. Отправить ход
async function sendMove(from, to) {
  await bridge.call('sendMessage', {
    channelId,
    text: `[MOVE:${JSON.stringify({from,to})}]`
  });
  // Сохранить состояние доски
  await bridge.call('storeSet', {
    key: 'board',
    value: getCurrentBoard()
  });
}

Ограничения

Важно: Мини-приложения запускаются в iframe с ограниченным sandbox. Не все браузерные API доступны.
  • Размер данных в хранилище: не более 64 КБ на ключ, 1 МБ суммарно на канал
  • Сообщения через sendMessage: не чаще 1 раза в секунду
  • Доступ к камере, микрофону, геолокации — запрещён
  • Отправка сообщений требует верифицированного аккаунта
  • Мини-апп работает только пока его окно открыто
  • subscribe() требует прямого вызова через window.parent.TES3API
